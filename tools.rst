=====
Tools
=====

In order to use Sesam to its full potential we have designed several tools that can help you be as efficient and effective as possible. These tools are helpful both in terms of data governance and data insight, but also in terms of developing integrations and performing master data management. 

In this section you learn both what these tools are, how they work and how we can use them effectively.
