Data Management
===============

.. toctree::
   :maxdepth: 1
   :hidden:

   Enrich <tutorial-getting-started-enrich>
   Connect <tutorial-getting-started-connect>
   Transform <tutorial-getting-started-transform>

In this section we we will cover the data management steps in a Sesam project. Data management in Sesam cover mapping imported data to the :ref:`global model <whatis-global>`, connecting this data to other imported data and :ref:`merging <merging_feature>` them. The final phase of data management in Sesam is the late schema binding, where we map global properties to the schema of the target system.
