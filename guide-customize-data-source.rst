This guide displays the tutorials available to create and customize your own :doc:`Custom Data Source <custom-data-source>`. 

.. admonition::  Objectives:
   
    After you complete these tutorials you would have learned how to:

    #. Customize a data source
    #. Adapt pipes in Sesam to connect to the data source
    #. Add continuation support to Microservice and its associated pipe
