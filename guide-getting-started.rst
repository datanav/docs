Getting started
===============

.. toctree::
   :maxdepth: 1
   :hidden:

   Setting up our Sesam instance  <tutorial-signup>
   Uploading a baseline configuration  <tutorial-upload-baseline-config>

In order to start using Sesam, you will need to sign up, create an instance and upload a baseline configuration. 
This step-by-step guide will help you getting started.

.. admonition::  Good to know
   
   After you complete these tutorials you would have:

   #. Signed up for Sesam
   #. Connected to a subscription
   #. Uploaded a baseline configuration
