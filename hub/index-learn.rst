===========
Learn Sesam
===========

Learn Sesam contains course descriptions, lessons and tutorials organised as guides. If you have not used Sesam before, we recommend that
you start by following the :doc:`Getting started guide <guide-getting-started>` in addition to taking our :doc:`Sesam Training Course <sesam-training-course>`.
