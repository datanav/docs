Data Synchronization
====================

.. toctree::
   :maxdepth: 1
   :hidden:

   Collect <tutorial-getting-started-collect>
   Share <tutorial-getting-started-share>

In this section we we will cover the data synchronization steps in a Sesam project. Data synchronization in Sesam is cover the import and export of data in and out of the Sesam hub. There are many :ref:`best practices <best-practice-workflow>` and :ref:`data synchronizaiton features <data-synchronization-features>` we need to consider for both importing and exporting data. The following tutorials will cover the most basic patterns we use. 

.. admonition::  Objectives:
   
   After you complete these tutorials you will have:

   #. Created a Collect pipe
   #. Created a Share pipe
   #. Seen the data synchronized into HubSpot
